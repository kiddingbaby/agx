# v0.1.0

> Initial baseline release. · 中文：[v0.1.0.zh.md](v0.1.0.zh.md)

The first tagged release. Establishes agx as a single-user CLI that
manages OpenAI-compatible relay profiles and syncs them into `codex` /
`claude` / `gemini` / `opencode` under isolated managed contexts.

## Surface

- **Profile lifecycle** — `agx add` / `edit` / `rm` / `ls` / `show` /
  `use` / `current`. `base_url` + `api_key` required; `model` optional
  (required when launching `opencode`).
- **Launcher** — `agx run <agent> [profile] [-- <native args>]` and the
  per-agent aliases `agx codex` / `claude` / `gemini` / `opencode`. Each
  agent runs in `~/.config/agx/contexts/<agent>/<name>/` without
  touching host-level `~/.codex`, `~/.claude`, `~/.gemini`,
  `~/.config/opencode`.
- **Profile resolution priority** — positional `profile` >
  `AGX_PROFILE` env > `agx current`. Pairs with `direnv` for
  per-directory pinning; never flips the global default.
- **Backup / restore** — `agx backup <agent>` snapshots the current
  managed target; `agx restore <agent>` rolls back the latest snapshot
  (`codex` / `claude` / `gemini`).
- **Opt-in auto-backup** — `AGX_AUTO_BACKUP=1` snapshots before each
  `agx run`.
- **Diagnostics** — `agx doctor` reports issues with suggested
  remediation; every emitted `code` is documented in
  [`docs/doctor-issues.md`](../doctor-issues.md), enforced by a
  bidirectional test against the runtime emitter.
- **Mutation guard** — every mutating entry point (add/edit/remove,
  managed profile flows, target API, binding helpers, backup/restore)
  captures profile YAML, state, and per-agent on-disk config before
  the change; any failure rolls back; panics still exit non-zero.
- **Shell completion** — `agx completion {bash|zsh|fish|powershell}`.
- **Version info** — `agx version` reports `version` / `commit` /
  `date` / `go` / `os` / `arch`.
- **First-run UX** — `agx` / `agx ls` with empty store print guidance.
- **Man pages** — 19 man pages shipped with releases; Homebrew formula
  installs them automatically.

## Distribution

- **Homebrew tap** — `brew install kiddingbaby/agx/agx`.
- **Pre-built binaries** — Linux / macOS, amd64 / arm64.
- **`go install`** — `go install github.com/kiddingbaby/agx/cmd/agx@latest`.
- **Source build** — Go 1.24+, toolchain declared in `mise.toml`.

## Stability

Pre-1.0. The public CLI surface is documented in
[`docs/cli-contract.md`](../cli-contract.md) and the compatibility
policy in [`COMPATIBILITY.md`](../../COMPATIBILITY.md). Internal state
fields and file layout may shift before 1.0 — do not depend on them.
